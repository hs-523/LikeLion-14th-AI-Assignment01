# LikeLion-14th-Assignment-Template
🦁 SKHU 멋쟁이사자처럼 14기 과제 PR 템플릿 레포지토리입니다.

< 일상에서 사용하는 서비스 / 선택한 서비스 >
- 전방 충돌 방지 시스템(AEB)

< 이 서비스는 무엇을 예측하는지 >
- 차량에 장착된 카메라와 레이더 센서를 바탕으로 전방의 물체, 차량, 보행자 등을 인식한다. 이후 차량의 속도와 거리 정보를 바탕으로 충돌 위험을 판단한다.

< 어떤 data가 사용될까(3가지 이상 작성) >
- 카메라 영상 데이터(차량 전방 이미지, 영상 정보로 차량 특정, 보행자, 물체 등을 인식)
- 레이더 데이터(전방 물체와의 거리를 측정하여 속도를 기반으로 충돌 위험 계산)
- 차량 주행 데이터(현재 속도, 액셀과 브레이크 상태 등이 정보 확인 / 운전자가 가속 혹은 감속 중인지 확인)

< 모델은 어떻게 학습하는가 >
- 차량의 카메라와 레이더를 활용하여 차량, 보행자, 물체 등의 정보를 직접 확인하고, 이후 다양한 주행 환경(주 야간, 악천후 등)을 반복적으로 학습하여 실제 도로에서 정확하게 작동하도록 학습한다.

< 이 모델이 더 좋아지려면 어떻게 해야 할까? >
- 모델의 성능을 높이기 위해서는 다양한 주행 환경 데이터를 보다 많이 확보하는 것이 중요하다. 카메라의 인식이 힘든 어두운 야간, 악천후 등 다양한 환경의 데이터를 학습할 수록 정확도가 향상된다.
- 무선 업데이트를 바탕으로 학습이 완료된 새로운 모델을 다른 차량들에게 빠르게 적용하는 것이 중요하다.
- 이를 바탕으로 한 기업이 테슬라다. 테슬라의 경우 지속적인 테스트와 무선 업데이트를 바탕으로 출시가 지난 차량들도 현재 출시된 차량처럼 뛰어난 성능을 지닌 것을 확인할 수 있다.
